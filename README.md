# 神奇寶貝碰碰碰 (PokMatch)

一個消除類遊戲。

## 🎮 遊戲特性

- **經典Match-3玩法**: 交換相鄰方塊消除
- **漸進式難度**: 6×6 → 8×8 → 10×10 網格
- **徽章系統**: 20個不同的館主和徽章
- **電源收集**: 點擊金/紅色閃光方塊觸發超級技能
- **博士幫助**: 每三關獲得一次救命機會
- **完整音效**: B遊戲音效
- **響應式設計**: 支援手機和電腦版本

## 📝 遊戲規則

1. **交換**：點擊兩個相鄰的交換位置
2. **消除**：形成3個或以上相同的水平/垂直線
3. **連鎖**：自動消除會累積連鎖倍率，增加得分
4. **特殊技能**：點擊金/紅色方塊觸發十字或星形消除
5. **時間挑戰**：每關限時60秒

## ⏱️ 計分

- 基礎得分 = 消除方塊數 × 10 × 連鎖倍率
- 目標分數逐級增加
