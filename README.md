# 神奇寶貝碰碰碰 (Pokémon Match-3 Game)

一個完整的寶可夢主題的消除類遊戲，所有資源已本地化部署。

## 🎮 遊戲特性

- **經典Match-3玩法**: 交換相鄰方塊消除3個以上相同寶可夢
- **漸進式難度**: 6×6 → 8×8 → 10×10 網格
- **道館徽章系統**: 20個不同的道館館主和徽章
- **電源收集**: 點擊金/紅色閃光方塊觸發超級技能
- **大木博士幫助**: 每三關獲得一次救命機會
- **完整音效**: BGM + 寶可夢叫聲 + 遊戲音效
- **響應式設計**: 支援手機和電腦版本

## 📝 遊戲規則

1. **交換**：點擊兩個相鄰的寶可夢交換位置
2. **消除**：形成3個或以上相同寶可夢的水平/垂直線
3. **連鎖**：自動消除會累積連鎖倍率，增加得分
4. **特殊技能**：點擊金/紅色方塊觸發十字或星形消除
5. **時間挑戰**：每關限時60秒

## ⏱️ 計分

- 基礎得分 = 消除方塊數 × 10 × 連鎖倍率
- 目標分數逐級增加
