邏輯益智網頁遊戲

這是一個基於瀏覽器運行的輕量級益智挑戰遊戲。本專案旨在展示網頁前端互動邏輯、資源管理與部署流程。

專案目錄結構
/
├── index.html          # 遊戲主入口
├── assets/             # 視覺資源
│   ├── pokemon/        # 角色素材 (已替換為通用資產)
│   ├── badges/         # 進度成就徽章
│   └── leaders/        # 關卡守關者頭像
└── audio/              # 音訊資源 (背景音樂與音效)

使用瀏覽器（Chrome, Edge, Firefox）直接開啟 index.html 即可執行。

線上連結
本專案已部署於 GitHub Pages，可直接透過以下連結進行體驗：
https://ssiet2500-star.github.io/pok/

技術說明
資源管理：採用相對路徑引用，確保跨平台部署的相容性。

音訊控制：為解決瀏覽器自動播放限制，採用點擊觸發式邏輯（User-Interaction Trigger），確保聲音能正常載入。

部署環境：透過 GitHub Pages 進行持續部署。

授權與維護
本專案僅作為個人前端技術實踐與邏輯控制測試之用途。若需擴充新功能，請參考 index.html 中的 JS 區塊進行邏輯調整。
